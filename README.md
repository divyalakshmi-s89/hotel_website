# 🍛 Jeya Hotel – Full Stack Restaurant Ordering Platform

A modern full-stack restaurant website built for **Jeya Hotel**, offering authentic home-style South Indian food with a seamless online ordering experience.

The platform allows customers to browse menus, place orders, contact the hotel through WhatsApp, and submit festival or bulk-order enquiries. Orders are securely stored in MongoDB and can be managed through an admin dashboard.

---

## ✨ Features

### Customer Features

* 🍽️ Interactive Food Menu
* 🛒 Add to Cart & Order Placement
* 📱 WhatsApp Order Notifications
* ⭐ Customer Reviews & Feedback
* 🎉 Festival & Bulk Order Enquiries
* 📍 Contact Information & Google Maps Integration
* 💬 Floating WhatsApp Support Button
* 📲 Mobile-Friendly Responsive Design

### Admin Features

* 👨‍💼 Admin Dashboard
* 📦 Order Management
* 📊 Customer Order Tracking
* 🗄️ MongoDB Database Integration
* 🔔 Real-Time Order Notifications

## 🚀 Local Development

### Backend Setup

```bash
cd backend
npm install
npm run dev
```

Backend runs on:

```text
http://localhost:5000
```

### Frontend Setup

```bash
cd frontend
npm install
npm start
```

Frontend runs on:

```text
http://localhost:3000
```

---



### Frontend

```env
REACT_APP_API_URL=http://localhost:5000
```

---

## 🚀 Deployment

### Frontend Deployment (Vercel)

```bash
cd frontend
npm run build
```

Deploy the frontend project on Vercel and configure the production API URL.

### Backend Deployment (Render)

```bash
cd backend
npm start
```

Add all required environment variables in the Render dashboard before deployment.
